Read the `README.md` file within this skill, and explore the other available files.

Key files:
- `README.md` — brand context, content + visual foundations, iconography, and a full index.
- `colors_and_type.css` — design tokens (colors, gradients, type, radii, shadows). Start here; link it in any HTML you build.
- `assets/` — logos, the 8# mark, campus photography, placeholder avatar.
- `ui_kits/website/` — high-fidelity, reusable React + CSS components recreating the 8bit marketing site. The best reference for assembling real pages.
- `preview/` — small specimen cards illustrating each token/component.

If creating visual artifacts (slides, mocks, throwaway prototypes, etc.), copy assets out and create static HTML files for the user to view. If working on production code, copy assets and read the rules here to become an expert in designing with this brand.

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some clarifying questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

Brand shorthand: playful, rounded, multi-color (teal-led) on white; **Nunito** throughout (Black/ExtraBold for display, semibold for body); pill buttons & badges; `rounded-3xl` cards on soft "pop" shadows; full-width color/gradient section bands joined by wavy dividers; floating blurred blobs; gentle fade/slide motion. No emoji. Spell **Mānoa** with the macron.
